How these were built:
1) Started from generated L80 preset matrix files.
2) Crawled public KissAssist .ini Library entries and selected a strong class profile per class.
3) Merged richer combat sections (Buffs/DPS/Heals/etc.) from KA source into each class preset.
4) Re-applied Solo/Group/Raid movement/core overlays for consistency.
5) Removed forum URL tags from imported lines.

Important:
- KA sources are mostly high-level (often 115-130), not true level 80.
- Spell/disc/item names may need downgrades/edits for your server and level.
- See KAHYBRID_SOURCES.txt for exact class-to-source mapping.
