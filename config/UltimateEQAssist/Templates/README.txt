UltimateEQAssist class pack generated.
Files: UltimateEQAssist_<CLASS>.ini
Classes: WAR CLR PAL RNG SHD DRU MNK BRD ROG SHM NEC WIZ MAG ENC BST BER

Usage:
1) Copy desired template to your character ini naming pattern, for example:
   UltimateEQAssist_${EverQuest.Server}_${Me}_${Me.Level}.ini
2) In MQ2Mule / UltimateEQAssist UI, set INI field to that name and click Import.
3) Fill/adjust BuffCond and spell names per server spell data.
